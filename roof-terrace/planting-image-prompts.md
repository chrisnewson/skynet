## How to use these prompts

**Midjourney v6** — Append `--ar 16:9 --v 6 --style raw --s 250` for cinematic landscape framing.

**DALL·E 3 (ChatGPT)** — Paste as-is; add "wide landscape format" if needed.

**Stable Diffusion / Adobe Firefly** — Paste as-is; consider adding "8k, photo-realistic, professional garden photography, golden hour" as quality boosters.

**Tips:**
- Run each prompt 4 times to get variation; pick the best.
- If a particular plant looks wrong, the AI often misidentifies; add explicit cues like *"Phormium tenax — strap-shaped sword leaves in upright fan, 1.5m tall, bronze-purple colour"*.
- For consistency across the 12 images, use the same AI model and tweak the consistent-setting preamble.
- Mid-summer afternoon golden hour was chosen for visual appeal — winter shots would show much sparser planting (Cynara dies back, Tamarix bare).
